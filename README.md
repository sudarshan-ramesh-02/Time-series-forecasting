Time series forecasting using Facebook Prophet

Time series forecasting deals with making predictions or 'forecasting' when the feature data is subject to change over time (ie) stock prices, weather, etc. Facebook Prophet is a python library that helps with predictions and forecasting when the decisive feature is date and time

Dataset deals with the history of 'value' of objects on particular days.
Forecasts were made on the values of the objects for the next hundred days and the trends of of various of forecast were observed.
